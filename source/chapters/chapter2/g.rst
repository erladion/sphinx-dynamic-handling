Troubleshooting
===============

If you run into common issues, consult this guide before raising a ticket.

**Issue 1: Component fails to start.**
* **Solution:** Check the `error.log` file for dependency issues.

**Issue 2: Slow performance.**
* **Solution:** Increase the available memory for the service or reduce the number of concurrent connections.

Need more help? Contact our support team.
